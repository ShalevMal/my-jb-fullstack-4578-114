פקודות Git & GitHub חיוניות

1. הגדרות בסיסיות למשתמש ב-GitHub

git config --global user.name "Your Name" - הגדרת שם המשתמש ב-GitHub
git config --global user.email "your-email@example.com" - הגדרת כתובת האימייל שלך ב-GitHub

2. יצירת מאגר Git חדש

git init - יצירת מאגר Git חדש בתיקייה הנוכחית

3. חיבור למאגר מרוחק (GitHub)

git remote add origin https://github.com/username/repo.git - חיבור המאגר המקומי שלך למאגר מרוחק ב-GitHub

4. הוספת קבצים לאזור ההמתנה

git add <filename> - הוספת קובץ מסוים לאזור ההמתנה
git add . - הוספת כל הקבצים החדשים/שונו בתיקייה הנוכחית

5. ביצוע Commit לשינויים

git commit -m "Your commit message" - ביצוע Commit לקבצים המוכנים עם הודעה תיאורית

6. דחיפת השינויים למאגר מרוחק (GitHub)

git push -u origin <branch-name> - דחיפת הסניף הנוכחי ל-GitHub
git push - דחיפת השינויים האחרונים שבוצעו ל-GitHub

7. משיכת שינויים ממאגר מרוחק (GitHub)

git pull - משיכת ומיזוג שינויים מהמאגר המרוחק לסניף הנוכחי

8. יצירת סניף חדש

git branch <new-branch-name> - יצירת סניף חדש
git checkout -b <new-branch-name> - יצירת סניף חדש ומעבר אליו

9. מעבר בין סניפים

git checkout <branch-name> - מעבר לסניף קיים

10. מיזוג סניפים

git merge <branch-name> - מיזוג הסניף הנבחר לתוך הסניף הנוכחי

11. בדיקת סטטוס המאגר

git status - הצגת מצב התיקייה הנוכחית ואזור ההמתנה

12. הצגת היסטוריית הקומיטים

git log - הצגת היסטוריית כל הקומיטים בסניף הנוכחי

13. ביטול שינויים

git reset <file> - הסרת קובץ מאזור ההמתנה
git checkout -- <file> - שחזור שינויים לקובץ (ביטול שינויים מקומיים)

14. מחיקת סניף

git branch -d <branch-name> - מחיקת סניף מקומי

15. שכפול מאגר מרוחק (GitHub)

git clone https://github.com/username/repo.git - שכפול מאגר GitHub למחשב המקומי

16. משיכת ומיזוג סניפים עם המאגר המרוחק

git fetch - משיכת כל השינויים מהמאגר המרוחק
git pull - משיכת ומיזוג שינויים למאגר המקומי

17. הצגת רשימת סניפים

git branch - רשימת סניפים מקומיים
git branch -r - רשימת סניפים מרוחקים
git branch -a - רשימת סניפים מקומיים ומרוחקים

18. תהליך עבודה לסיום עבודה על סניף

git push origin <branch-name> - דחיפת הסניף ל-GitHub
git checkout main - מעבר לסניף main
git merge <branch-name> - מיזוג הסניף שהושלם ל-main
git branch -d <branch-name> - מחיקת הסניף (במקומי)

19. עבודה עם תגיות (גרסאות)

git tag <tag-name> - יצירת תג חדש (בדרך כלל לסימון גרסאות)
git push origin <tag-name> - דחיפת התג ל-GitHub
