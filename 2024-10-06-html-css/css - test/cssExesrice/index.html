מהם CSS Selectors?

CSS Selectors הם כלים המאפשרים לבחור אלמנטים בדף HTML כדי לעצב אותם. באמצעות selectors, ניתן לשלוט במראה של אלמנטים שונים על פי תכונותיהם.

ה-selector הראשון שייבחר בדף הוא זה שיגרום לשינוי במראה האלמנט. אם מספר selectors מתאימים לאותו אלמנט, ה-selector הראשון שיגיע בקוד הוא זה שיתקבל.

ישנם סוגים שונים של CSS Selectors, וכל אחד מהם משמש למטרות שונות בהתאם לצורך בעיצוב.

סוגים של CSS Selectors

  • Selector לפי תגית: בוחר אלמנטים לפי שם התג. לדוגמה, p בוחר את כל תגי הפסקה.
  • Selector לפי class: בוחר אלמנטים בעלי class מסוים. לדוגמה, .my-class בוחר את כל האלמנטים עם class בשם "my-class".
  • Selector לפי id: בוחר אלמנט עם id מסוים. לדוגמה, #my-id בוחר את האלמנט עם id בשם "my-id".
  • Attribute Selector: בוחר אלמנטים לפי תכונת attribute. לדוגמה, [type="text"] בוחר את כל האלמנטים עם attribute type ששווה ל-"text".
  • Combinators: משתמשים ביחסים בין אלמנטים כדי לבחור אלמנטים. לדוגמה, div p בוחר את כל הפסקאות שנמצאות בתוך תגי div.
  • Pseudo-classes: בוחרים אלמנטים לפי מצבם. לדוגמה, a:hover משנה את צבע הקישור כאשר העכבר עובר עליו.

דוגמאות קוד

Selector לפי תגית

p {
    color: blue;
    font-size: 16px;
}

דוגמה זו משנה את צבע הפסקאות לכחול.

Selector לפי class

.my-class {
    background-color: yellow;
    padding: 10px;
}

במקרה זה, כל אלמנט עם class "my-class" יקבל רקע צהוב.

Selector לפי id

#my-id {
    font-weight: bold;
}

במקרה זה, האלמנט עם id "my-id" יהיה מודגש.

Attribute Selector

[type="text"] {
    border: 1px solid black;
}

דוגמה זו מוסיפה גבול שחור לכל האלמנטים עם attribute type ששווה ל-"text".

Combinator Selector

div p {
    margin: 0;
}

דוגמה זו מסירה את המרג'ין מהפסקאות שנמצאות בתוך תגי div.

Pseudo-class Selector

a:hover {
    color: red;
}

הקישור יהפוך לאדום כאשר העכבר עובר עליו.
